# BASIS Architecture

This repository contains architecture papers, threat models, diagrams, ADRs, and research documents related to the BASIS (Building Automation Secure Identity Service) initiative.

BASIS explores identity-aware authorization, centralized policy enforcement, and auditability for operational technology (OT) environments such as:

- data centers
- hospitals
- campuses
- warehouses
- commercial buildings
- industrial facilities
